# RESUME-DEV

Desktop-inspired developer portfolio starter (dark, tech-first UI with a social icon rail).

## Run locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Personalize

- Replace placeholder image URL in `index.html` with your own photo.
- Update social links in `index.html`:
  - LinkedIn: `https://www.linkedin.com/in/your-username`
  - GitHub: `https://github.com/your-username`
  - X: `https://x.com/your-username`
  - Instagram: `https://instagram.com/your-username`
- Edit headline/tagline/project cards in the hero section.
